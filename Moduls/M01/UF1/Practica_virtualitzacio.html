* Observació: posa en lloc de <nomusuari> el nom del teu usuari (és el què apareix al terminal abans de la @)

13.- Troba informació de què és AWS de Amazon.

Es una empresa que actualment pertany a amazon, el seu objectiu es proporcionar maquines virtuals de forma totalment online, tens que pagar una cuota per fer servir una.

  A quin servei ens referim quan parlem de EC2?

Es una part de la plataforma d’amazon de les maquines virtuals, aquest permet que els usuaris executin equips virtuals i tenir les seves aplicacions.

14.- Entra a AWS Educate amb el teu usuari que se t’ha subministrat. Crea una màquina virtual amb Ubuntu Server 20.04LTS.

Instal·la el servidor nginx i crea una pàgina web teva que mostri un Hola sóc … (seguit del teu nom).

Pots veure com fer-ho en aquest vídeo: AWS Educate, ubuntu 18.04 i apache2

* Nota 1: Fixa’t que al vídeo s’instal·la apache mentre que nosaltres ho fem amb nginx.

* Nota 2: En comptes de vi com diu el vídeo, pots fer servir nano.

En el meu cas no l’he pogut fer pero explicare els passos que tenim que fer per aconseguir el resultat desitjat.

Primer de tot ens tenim que crear el compte a la pagina d’AWS. Una vegada tinguem el compte tenim que entrar per crear una maquina virtual, en el nostre cas agafarem l’ubuntu server, agafarem totes les opcions basiques per crear la nostra maquina virtual. Una vegada elegit tot tindrem que crear el nostre certificat, una vegada fet aixo tindrem un boto on ens diu que anem a les instancies, li tenim que donar a aquell boto per anar al panell de control de la nostre maquina virtual. Seguidament tenim que utilitzar el programa PuTTY Key per transformar el nostre certificat en un format ppk, ara tenim que agafar la ipv4 de la nostre maquina virtual d’amazon i posarla en el programa PuTTy, ara tenim que seleccionar el certificat que tenim en ppk, si ens detecta tot be li donarem a open i entrarem a la nostre maquina virtual. Ens demanara un usuari, per a ubuntu el usuari es ubuntu. Ara tenim que anar al panell de la maquina virtual i tenim que anar a l’apartat dels security group i crearem un nou amb les seguents normes d’entrada: Custom TCP, port 80 i 443 (Aixo seveix per obrir aquests ports), una vegada fet tindrem que anar a instancies i tindrem que activar el nou seecurity group.

A continuació anem a instalar el nginx, per instalarlo tenim que ficar la seguent comanda:

sudo apt install nginx.

Una vegada fet tot aixo tenim que ficar la seguent comanda:

sudo nano /var/www/html/index.html, aqui podrem escriure per editar la nostre pagina.

Una vegada fet aixo nomes tindrem que agafar el link i anar a la pagina per poder veure els canvis.

15.- Troba informació de què és Docker?

Es una aplicació que implementa una API d’alt nivell per proporcionar contenidors que executen procesos de manera aillada

 En què es diferencia de serveis de virtualització com VirtualBox, VMWare, EC2, ...?

Aquest programa no li cal un sistema operatiu independent, es basa en les funcionalitats del nucli i utilitza l’aillament dels recursos
